# How to setup VScode

You just need to install the .deb package.

```bash
wget https://code.visualstudio.com/sha/download?build=stable&os=linux-deb-x64
sudo dpkg -i 
```



Learn more at [https://code.visualstudio.com/](https://code.visualstudio.com/)
